# Land Management System (Django)

Web application for managing land parcels and lease information.

## Features

- Land parcel database
- Filtering and search
- Pagination
- AJAX requests
- Map visualization with Leaflet
- User authentication
- Export and import data

## Technologies

- Python
- Django
- PostgreSQL
- JavaScript
- Bootstrap
- Leaflet.js
- AJAX

## Description

This project is a web application designed to manage land parcels, lease agreements and related geographic data.

The system allows users to:

- store land parcel information
- filter and search records
- visualize parcels on an interactive map
- manage users and permissions
